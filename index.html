Bare Bones

One heck of a css flex-based grid system for developing responsive projects

What Is It?

Bare Bones is a small collection of CSS files that can help you rapidly develop sites that look beautiful at any size, be it a 17" laptop screen or an iPhone. Bare Bones is built on three core principles:

Sturdy flex grid

Using CSS's flex property gives you the ability of a flex item to alter its dimensions to consume the space around it. No more float-based layouts!

Responsive down to mobile

Targeting the unpredictability of screen sizes, the row and column structure is a fully responsive layout, looking perfect on all devices.

Lightweight, rapid development

Fitting snug in CSS file with less than 100 lines, the simple class name convention and percent-based columns make for lightning fast development.

Getting Started

Bare Bones is so light-weight and quick to implement into any new or existing website. All you need is the parent wrapper class, one or more child rows, and within the rows, any number of columns that equate to 100. Each column class is prefixed with a "flex-c-" abbreviated for flex column and finally concatenated with its percent-based width.

Media Queries

Bare Bones uses a grand total of 3 media queries to construct the scalable grid. They have been placed atf 1200px, 992px, and 768px respectively. This sets up an ideal situation for a reliable and consistent website that scales effortlessly from desktop down to mobile. Bare Bones targets media queries using a min width specifier and only modifies that parent flex-wrap class. So while the main container is adjusting to the viewport, the percent-based columns respond wonderfully to their parent.
